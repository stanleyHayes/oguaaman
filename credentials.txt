 PRODUCTION (free tiers) — Render + Vercel + Atlas
------------------------------------------------------------------------------
Database   MongoDB Atlas M0 (free) — cluster0.pioeoms.mongodb.net, db: oguaa
           URI lives in backend/.env (gitignored) and in Render env MONGODB_URI.
           Atlas → Network Access must allow 0.0.0.0/0 (Render has no fixed IPs).
           RESEED (destructive: drop + insert):  cd backend && go run ./cmd/seed

API        Render Blueprint (render.yaml at repo root) — Docker, free plan,
           region Frankfurt. Service: oguaa-api → https://oguaa-api.onrender.com
           Health check: GET https://oguaa-api.onrender.com/api/health
           Deploy: Render dashboard → New → Blueprint → select this repo;
           Render builds backend/Dockerfile and prompts once for the
           sync:false values (MONGODB_URI, PAYSTACK_SECRET_KEY, ANTHROPIC_API_KEY).

Portal     Vercel (free) — project root directory: frontend/  (frontend/vercel.json
           sets Vite build, SPA rewrites, security headers).
           URL: https://oguaa.vercel.app
           Deploy: import repo in Vercel → set Root Directory to "frontend" →
           add the env vars below → deploy. Previews work the same way.

Render env vars (oguaa-api):
  MONGODB_URI            mongodb+srv://… (Atlas — SECRET, prompted by Blueprint)
  MONGODB_DB             oguaa
  JWT_SECRET             auto-generated by Blueprint (generateValue)
  AUTH_REQUIRED          true
  ALLOWED_ORIGIN         https://oguaa.vercel.app   (comma-separate to add domains)
  PUBLIC_PORTAL_URL      https://oguaa.vercel.app   (Paystack callback base)
  PUBLIC_API_URL         https://oguaa-api.onrender.com
  UPLOAD_DIR             /app/uploads
  PLATFORM_FEE_PERCENT   5
  PAYSTACK_SECRET_KEY    (blank = payments run in labelled SIMULATION)
  ANTHROPIC_API_KEY      (blank = AI writing bar runs in labelled simulation)

Vercel env vars (frontend):
  VITE_API_URL                  https://oguaa-api.onrender.com
  VITE_CLOUDINARY_CLOUD_NAME    (unsigned-upload Cloudinary cloud)
  VITE_CLOUDINARY_UPLOAD_PRESET (its unsigned preset)

Free-tier caveats:
  • Render free sleeps after ~15 min idle — first hit after a nap takes ~50s.
  • Render free disk is EPHEMERAL: /uploads files vanish on redeploy/restart.
    Durable member images must go through Cloudinary (portal already does).
  • Only $PORT is exposed publicly — the gRPC server (50051) is NOT reachable
    from the internet on Render; the portal uses REST + GraphQL only.
  • Vercel preview URLs are NOT in ALLOWED_ORIGIN — add them (or a custom
    domain) to ALLOWED_ORIGIN on Render if you need previews to call the API.
  • If the Vercel project name differs, update ALLOWED_ORIGIN / PUBLIC_PORTAL_URL
    on Render AND the canonical/OG/sitemap URLs in frontend/ (index.html,
    public/robots.txt, public/sitemap.xml) — all currently say oguaa.vercel.app.
