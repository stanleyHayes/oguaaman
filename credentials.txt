 OGUAA — credentials & access (local / development)
==============================================================================
Auth is passwordless: sign in with a phone or email and a one-time code (OTP).
In dev (AUTH_REQUIRED unset/false) the backend RETURNS the code in the API
response and the UIs display it — no SMS is sent. A real SMS/WhatsApp provider
(e.g. Hubtel) plugs into the OTPSender interface in production.
